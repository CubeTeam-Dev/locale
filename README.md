# CubeTeam Locale

CubeTeamは、より多くのユーザーにサービスを届けるため、多言語対応を積極的に進めています。
このリポジトリは、CubeTeamプロジェクトで使用される翻訳ファイルを管理するためのものです。

## 翻訳への参加方法

CubeTeamでは、常時**翻訳者**を募集しています。
ご興味がありましたら、以下のCrowdinプロジェクトからご参加ください。

- Crowdin プロジェクトページ: [https://crowdin.com/project/cubeteam](https://crowdin.com/project/cubeteam)

Crowdinアカウントを作成した上で、参加したい言語を選択し、各種テキストの翻訳・レビューにご協力ください。

## 対応言語について

現在利用可能な言語はCrowdin上の「Languages」一覧から確認できます。
進捗状況や翻訳量もCrowdin上でリアルタイムに確認可能です。

### あなたの言語がありませんか？

もしあなたが使用したい言語がCrowdinプロジェクトに存在しない場合は、ぜひ新しい言語の追加を提案してください。
提案は、以下のいずれかの方法で受け付けています。

- CubeTeam公式 [Discord サーバー](https://discord.gg/Se5QqQwj4h)での提案  
- GitHub Issue でのリクエスト

## ライセンスと利用ポリシー

翻訳ファイルのライセンスや利用ポリシーは、本リポジトリのライセンスファイルに従います。
翻訳に参加された場合、Crowdin上での利用規約および本リポジトリのライセンスへの同意が前提となります。
